使用的软件

🖥️ 操作系統

  • Windows 10 / Windows 11 - 仍然是最多人使用的个人操作系统

🌏 浏览器

  • Edge - 可以方便结合微软生态的浏览器

📝 笔记软件 / 文字编辑器

  • Typora - Markdown 编辑器

🧰 电脑工具

  • 图吧工具箱 - 入门最方便的电脑工具箱
显卡超频/监控工具
  • Afterburner - 用于拉显卡风扇曲线

➗ 科学公式编辑器

  • MathType - 在文档和演示文稿中插入数学公式、符号和矩阵

📌 截图工具

  • QQScreenShot - 从旧版本QQ中提取的截图工具，便携好用

🖼 图床平台管理工具

  • PicList - 继承自PicGo，开源免费的图床工具

💻远程控制工具

  • RayLink - 缺乏自主搭建服务器能力时的选择

🎮 Game Engine

  • Godot Engine - 免费开源的 2D 与 3D 游戏引擎

🦹🏻‍♂️ 反编译工具

  • IDA Pro - 功能强大、广泛应用于软件逆向工程的交互式反汇编工具
  • dnSpy - .NET 调试器和程序集编辑器
  • Il2CppDumper - 用于Unity游戏il2cpp模式下的逆向工程工具

🔎 OCR(从图片中识别并提取文字工具) / 翻译工具

  • 网易有道翻译 v10.1.0 - 成熟的OCR商业化软件

  • Umi-OCR - 开源、免费、离线的OCR软件

⚡解压软件

  • 7-Zip - 开源免费、无广告的解压软件

📹 视频录制 / 直播推流

  • OBS Studio - 开源免费的视频录制和直播推流软件

网络工程/嵌入式工具

  • ATK-XCOM.exe - 正点原子串口调试工具

  • SecureCRT - 终端仿真软件

  • MobaXterm - 远程终端控制工具

  • WireShark - 开源免费网络抓包工具
