八、常见问题

Q: 发布时提示包名已存在怎么办？

A: 改个独特的名字，比如 hexo-migrate-cli-yourname

Q: 我想在本地测试，不发布到 npm 可以吗？

A: 当然！使用 npm link 就可以在本地全局使用了

Q: 如何撤销发布的包？

A: npm unpublish hexo-migrate-cli --force（但不建议，超过 24 小时就不能撤销了）
